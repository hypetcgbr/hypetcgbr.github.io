# hypetcg preview landing page

Landing page estática para pré-lançamento da hypetcg no GitHub Pages.

## Estrutura

```txt
index.html
assets/
  css/style.css
  js/main.js
```

## Como publicar no GitHub Pages

1. Crie ou use o repositório `hypetcg.github.io`.
2. Envie estes arquivos para a branch principal.
3. Em `Settings > Pages`, publique pela branch principal, pasta `/root`.
4. O site ficará disponível em `https://hypetcg.github.io/`.

## O que trocar antes de divulgar

- Link do WhatsApp: procure por `https://chat.whatsapp.com/SEU-LINK-AQUI` no `index.html`.
- Link do Instagram: procure por `https://instagram.com/SEU-INSTAGRAM-AQUI`.
- Fotos reais: substitua a galeria temporária por imagens de boxes, cards PSA10 e acessórios.
- Domínio próprio depois: apontar `hypetcg.com.br` para GitHub Pages quando desejar.
